# `yoyoclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
yoyoclaw pairing list whatsapp
yoyoclaw pairing approve whatsapp <code> --notify
```
