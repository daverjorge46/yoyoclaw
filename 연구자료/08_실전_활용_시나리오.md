# 오픈클로(OpenClaw) 실전 활용 시나리오 30선

이 문서는 오픈클로의 다양한 스킬과 채널을 조합하여 실제 생활과 업무에 바로 적용할 수 있는 30가지 자동화 레시피를 담고 있습니다.

> **참고**: 모든 시나리오는 오픈클로 게이트웨이(`openclaw gateway run`)가 실행 중인 상태를 가정합니다.
> **문서 버전**: v1.1 (보강판)
> **저자**: 오픈클로 리서치팀

---

## 🚀 5분 만에 따라하기 (Quick Start)

가장 쉽고 빠르게 설정할 수 있는 베스트 시나리오 3선입니다.

### QS-1. 아침을 깨우는 모닝 브리핑 (초급)
**기능**: 매일 아침 7시, 날씨와 주요 일정을 텔레그램으로 브리핑합니다.
**소요 시간**: 약 5분
**준비물**: Telegram Bot Token, OpenWeatherMap API Key

1. **설정**:
   ```bash
   # 1. 텔레그램 봇 연결
   openclaw config set telegram.bot_token <YOUR_TOKEN>
   
   # 2. 날씨 API 키 설정
   openclaw config set weather.api_key <OPENWEATHER_KEY>
   ```

2. **자동화 등록**:
   ```bash
   openclaw schedule add "morning-brief" "0 7 * * *" --command "weather get --city Seoul | summarize '오늘 날씨와 옷차림, 주요 일정을 브리핑해줘' >> @telegram"
   ```

### QS-2. 유튜브 요약 비서 (초급)
**기능**: 긴 유튜브 영상 URL을 주면 핵심 내용만 3줄로 요약해줍니다.
**소요 시간**: 약 3분
**준비물**: 없음 (기본 LLM 사용)

1. **사용법**: 텔레그램이나 슬랙에서 봇에게 메시지 전송
   > "이 영상 요약해줘: https://youtu.be/..."

2. **원리**: `browser` 스킬이 자막을 추출하고 `summarize` 스킬이 요약합니다.

### QS-3. 개발자용 PR 리뷰어 (중급)
**기능**: GitHub에 Pull Request가 올라오면 자동으로 코드 리뷰를 남깁니다.
**소요 시간**: 약 10분
**준비물**: GitHub Token

1. **설정**:
   ```bash
   openclaw config set github.token <GITHUB_TOKEN>
   openclaw hook add --event github.pr_opened --command "coding-agent review --pr $EVENT_PAYLOAD"
   ```

## 1. 개인 생산성 (Personal Productivity)

### 1-1. 모닝 브리핑 봇 (Morning Briefing)
아침 기상 시간에 맞춰 날씨, 일정, 주요 뉴스를 요약해서 알려줍니다.

**선행조건**:
- [ ] OpenClaw Gateway 실행 중
- [ ] [OpenWeatherMap API](https://openweathermap.org/api) 키 발급
- [ ] [Telegram Bot](https://core.telegram.org/bots/api) 생성 및 토큰 확보

**관련 스킬**: `src/skills/weather`, `src/skills/summarize`

**설정 방법:**
1. 텔레그램 설정: `openclaw config set telegram.bot_token <TOKEN>`
2. 날씨 설정: `openclaw config set weather.api_key <KEY>`
3. 크론 등록:
   ```bash
   openclaw schedule add "morning-brief" "0 7 * * *" --command "weather get --city Seoul | summarize '오늘 날씨와 옷차림 추천해줘' >> @telegram"
   ```

**대화 예시:**
> **Bot**: ☀️ **오늘의 모닝 브리핑**
> - **날씨**: 서울 15°C, 맑음. 오후에 구름 조금. 가벼운 자켓 추천.
> - **뉴스**: 비트코인 1억 돌파, 애플 신제품 발표 등

**에러 처리:**
- ❌ `Error: weather skill not found` → `openclaw skills install weather` 실행
- ❌ `401 Unauthorized` → API 키가 올바른지 확인 (`openclaw config list`)
