- **패키지매니저**: pnpm (Bun도 지원)
- **빌드**: `pnpm build` (tsc)
- **린트**: `pnpm lint` (oxlint), `pnpm format` (oxfmt)
- **테스트**: `pnpm test` (vitest), `pnpm test:coverage`
- **커밋**: `scripts/committer "<msg>" <file...>`

- **확장 플러그인**: `extensions/*` 워크스페이스 패키지, `npm install --omit=dev`로 설치
- **Docker**: 공식 이미지 존재, non-root(`node` uid 1000) 실행
