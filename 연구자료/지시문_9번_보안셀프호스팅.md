## 임무
보안 설정, 프라이버시 보호, 셀프호스팅 심화 가이드를 작성해.

## 조사 범위 (각 항목당 최소 100개 신뢰 가능한 자료 수집)
1. **보안 아키텍처** - 오픈클로의 보안 모델, 데이터 흐름, 암호화 방식 (SECURITY.md 분석)
2. **API 키 관리** - 프로바이더 API 키 안전 관리법, 1Password 연동, 환경변수 관리
3. **네트워크 보안** - HTTPS 설정, 방화벽, 리버스 프록시(Nginx/Caddy/Traefik), VPN
4. **인증/접근제어** - DM 페어링, 허용 목록, 그룹 권한, 멀티유저 접근 제어
5. **데이터 프라이버시** - 대화 로그 저장 위치, 삭제 방법, AI 프로바이더로 전송되는 데이터
6. **로컬 전용 모드** - Ollama로 완전 로컬 운영, 외부 API 호출 없이 사용
7. **Docker 보안** - 컨테이너 격리, 네트워크 정책, 볼륨 권한, 샌드박스 모드
8. **셀프호스팅 인프라** - 가정용 서버, Raspberry Pi, NAS(Synology/QNAP), VPS 비교
9. **백업/복구** - 데이터 백업 전략, 자동 백업 스크립트, 재해 복구
10. **모니터링/로깅** - 로그 관리, 이상 탐지, 리소스 모니터링
11. **업데이트 보안** - 자동 업데이트 설정, 취약점 패치 관리
12. **GDPR/개인정보보호법** - 한국 개인정보보호법, GDPR 준수 시 고려사항
