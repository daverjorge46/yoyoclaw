## 배경
오픈클로(OpenClaw)는 구 몰트봇(MoltBot)/클로봇(ClawdBot)에서 리브랜딩된 오픈소스 멀티채널 AI 비서 플랫폼이야.

## 임무
설치부터 첫 메시지 송수신까지의 완전한 가이드를 작성해.

## 조사 범위 (각 항목당 최소 100개 신뢰 가능한 자료 수집)
1. **설치 방법 비교** - npm, Docker, Homebrew, Nix, 소스빌드 각각의 장단점
2. **OS별 설치 가이드** - macOS, Linux(Ubuntu/Debian), Windows(WSL2), Raspberry Pi
3. **최초 온보딩** - `openclaw onboard` 과정 상세 (각 단계 스크린샷급 설명)
4. **환경 설정** - .env 파일, 게이트웨이 설정, 포트 설정, 보안 설정
5. **AI 프로바이더 연결** - 모든 지원 프로바이더(Anthropic, OpenAI, Google, Ollama 등) API 키 설정법
6. **데몬/서비스 설정** - 백그라운드 실행, 자동시작, systemd/launchd 설정
7. **업데이트/마이그레이션** - 버전 업그레이드, 몰트봇→오픈클로 마이그레이션
8. **Docker 배포** - docker-compose.yml 상세, 볼륨 마운트, 환경변수
9. **클라우드 배포** - Fly.io, Railway, Render, DigitalOcean 각각의 배포 가이드
10. **초기 트러블슈팅** - 설치 시 흔한 에러와 해결법
