   ```bash
   # OpenClaw
   npm install -g openclaw && openclaw gateway run
   # LobeChat
   docker run -d -p 3210:3210 lobehub/lobe-chat

   docker compose up -d
   ```

5. **마이그레이션 가이드** - Dify→OpenClaw, n8n→OpenClaw, LobeChat→OpenClaw 전환 시 고려사항
