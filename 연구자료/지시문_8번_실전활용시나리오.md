## 임무

## 조사 범위 (각 항목당 최소 100개 신뢰 가능한 자료 수집)
1. **개인 생산성** (5개+)
   - 아침 브리핑 봇 (날씨+일정+뉴스 자동 요약)
   - 할일 관리 비서 (Notion/Todoist 연동)
   - 메모/일기 자동 정리 (Obsidian/Apple Notes)
   - 이메일 요약 비서
   - 독서 노트 정리
2. **업무/비즈니스** (5개+)
   - 고객 지원 챗봇 (FAQ 자동 응답)
   - 회의 요약 비서 (음성→텍스트→요약)
   - 프로젝트 상태 알림 (GitHub/Jira 연동)
   - 경쟁사 모니터링 (웹 스크래핑+요약)
   - 보고서 자동 생성
3. **개발자** (5개+)
   - 코드 리뷰 봇 (GitHub PR 자동 리뷰)
   - 에러 모니터링 알림 (로그 감시+Telegram 알림)
   - 문서 자동 생성 (코드→API 문서)
   - CI/CD 상태 봇
   - 기술 블로그 초안 작성
4. **콘텐츠 제작** (5개+)
   - 블로그 글 작성 보조
   - SNS 포스트 생성 (이미지+텍스트)
   - 번역 비서 (다국어 콘텐츠)
   - 유튜브 스크립트 작성
   - SEO 키워드 리서치 자동화
5. **스마트홈/IoT** (5개+)
   - 홈 오토메이션 (Home Assistant 연동)
   - 가계부 봇 (지출 기록+분석)
   - 건강 관리 봇 (운동/식단 추적)
   - 가족 공유 봇 (그룹채팅에서 가족 일정 관리)
   - 반려동물 관리 봇
6. **한국 특화 시나리오** (5개+)
   - 네이버 카페/블로그 모니터링
   - 배달앱 추천 봇
   - 한국 뉴스 요약 봇
   - 주식/코인 알림 봇
   - 공공데이터 활용 봇

## 각 시나리오 포함 내용
- 시나리오 이름과 설명
- 필요한 스킬/채널/프로바이더 목록
- 단계별 설정 방법
- 실제 대화 예시 (사용자↔봇)
- 크론/웹훅 설정 (자동화가 필요한 경우)
- 예상 월간 비용
- 난이도 (초급/중급/고급)
