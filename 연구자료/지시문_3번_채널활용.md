## 배경
오픈클로(OpenClaw)는 15개 이상의 메신저 채널을 지원하는 멀티채널 AI 비서야.
사용자는 Telegram과 카카오톡에서 사용하길 원함.
**중요: 카카오톡은 공식 지원 채널이 아님** - 대안/우회 방법 조사 필수.

## 임무
Telegram 완전 활용 가이드 + 카카오톡 연동 대안 + 기타 채널 비교를 작성해.

## 조사 범위 (각 항목당 최소 100개 신뢰 가능한 자료 수집)
1. **Telegram 봇 생성** - BotFather로 봇 생성, 토큰 발급, 봇 설정 상세
2. **Telegram 채널 연결** - openclaw에서 Telegram 연결 설정 전체 과정
3. **Telegram 고급 기능** - 그룹채팅, 멘션 게이팅, 스티커, 인라인 모드, 메시지 편집
4. **Telegram 미디어** - 이미지/음성/영상/파일 송수신
5. **카카오톡 연동 방안** - 공식 미지원이므로:
   - 카카오톡 봇 API (카카오 i 오픈빌더) 연동 가능성
   - 카카오톡 채널(구 플러스친구) + 웹훅 연동
   - n8n/Zapier 등 중간 브릿지 활용
   - 커스텀 확장 채널 개발 방법 (extensions/ 구조 참고)
   - 기존 카카오톡 봇 프레임워크와 연결
6. **채널 비교표** - 지원 15+개 채널 전체의 기능 비교 (미디어, 그룹, 봇API 등)
7. **멀티채널 라우팅** - 여러 채널 동시 운영, 에이전트별 채널 분리
8. **WebChat** - 브라우저 기반 채팅 UI 활용
9. **Discord 활용** - 서버 봇 설정, 슬래시 커맨드
10. **보안 설정** - DM 페어링, 허용목록, 그룹 권한
