## 보강 요구사항

### 각 시나리오(30개)에 추가해야 할 항목
1. **선행조건 블록** - 시나리오 실행 전 반드시 필요한 것들:
   ```
   ### 선행조건
   - [ ] OpenClaw Gateway 실행 중 (`openclaw gateway run`)
   - [ ] Telegram Bot Token 설정 완료
   - [ ] OpenAI API Key 설정 완료
   ```
2. **실제 CLI 명령어 검증** - `src/cli/`, `src/commands/` 코드를 확인하여 실제 존재하는 명령어만 사용. 존재하지 않는 명령어는 "예시 (향후 지원 예정)" 표기
3. **에러 처리 섹션** - 시나리오별 흔한 에러 1-2개와 해결법
   ```
   **흔한 문제:**
   - ❌ `Error: TELEGRAM_BOT_TOKEN not set` → `openclaw config set telegram.bot_token <TOKEN>`
   ```
4. **외부 서비스 링크** - 각 시나리오에서 사용하는 외부 서비스의 공식 문서/가입 링크
5. **예상 소요 시간** - 초기 설정에 걸리는 대략적 시간 (예: "초기 설정 약 15분")
6. **관련 OpenClaw 스킬 매핑** - 프로젝트 내 실제 스킬 파일 경로와 매핑

### 구조 개선
1. **난이도순 정렬** - 초급(10개) → 중급(12개) → 고급(8개) 순서로 재배치
2. **Quick Start 시나리오 3개** - 가장 쉽고 인상적인 3개를 문서 상단에 "5분 만에 따라하기"로 배치
3. **시나리오 간 연결** - "이 시나리오를 마쳤다면 → 다음 추천 시나리오" 가이드
