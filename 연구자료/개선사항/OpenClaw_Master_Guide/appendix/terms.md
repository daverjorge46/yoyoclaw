# 용어집 (Glossary)

## A

### A2A (Agent-to-Agent)
에이전트 간 직접 통신 방식. `sessions_send` API를 통해 메시지를 교환한다. 세나를 거치지 않고 빠르게 협업할 수 있다.

---

## C

### ClawHub
오픈크롤봇 스킬 공개 레포지토리 (예정). 스킬을 업로드/다운로드/검색할 수 있다.

### ContextTokens
모델의 컨텍스트 윈도우 크기. 입력과 출력 토큰의 총량을 제한한다. 너무 낮으면 맥락 상실, 너무 높으면 비용 폭증의 원인이 된다.

### CDP (Chrome DevTools Protocol)
브라우저를 프로그래밍 방식으로 제어하는 프로토콜. 오픈크롤봇의 브라우저 제어는 CDP를 기반으로 한다.

---

## G

### Gateway
오픈크롤봇의 제어평면. 채널 연결, 라우팅, 스케줄링, 이벤트 관리 등을 담당한다.

---

## M

### MCP (Model Context Protocol)
Anthropic의 툴 액세스 표준. 에이전트 간 자원 공유와 툴 액세스를 표준화한다.

### Memory
장기 기억 저장소. 벡터 검색(SQLite) + 키워드 검색(FTS5)을 지원한다. MEMORY.md에 저장된다.

---

## O

### OAuth
OpenAI/Anthropic 계정 연동 방식. API 키 없이도 계정으로 사용할 수 있다.

---

## P

### Pairing
보안을 위한 인증 방식. 알 수 없는 발신자만 수동 승인해야 한다.

---

## S

### Session
사용자와 에이전트 간의 대화 기록. JSONL 형식으로 저장된다.

### Skill
에이전트의 기능을 확장하는 패키지. 프롬프트 + 워크플로우 + 툴 + 자료로 구성된다.

---

## T

### Tailscale
안전한 원격 접속을 위한 VPN 서비스. Serve(Funnel)로 공개/사설망 노출을 제어한다.

### Temperature
모델의 창의성을 조절하는 설정. 낮을수록 일관적이고 높을수록 창의적이다. 기본 0.3-0.7, 창의적 작업만 0.8+.

---

## W

### Workspace
에이전트의 작업 디렉토리. AGENTS.md, SOUL.md, MEMORY.md 등 설정 파일이 저장된다.

---

## 기타 용어

### Agent
AI 모델이 실행되어 사용자 요청을 처리하는 엔티티.

### Batch (배칭)
유사 쿼리를 묶어서 처리하는 기법. API 호출 횟수를 줄여 20-30% 비용 절감이 가능하다.

### Cache Hit Ratio (캐시 히트율)
캐시에서 결과를 찾은 비율. 30-50%를 목표로 한다.

### Handoff-based
인계 기반 멀티에이전트 패턴. 한 에이전트가 작업을 완료하면 다음 에이전트로 넘긴다.

### Parallel execution
병렬 실행 멀티에이전트 패턴. 여러 에이전트가 동시에 작업한다.

### Sequential refinement
순차적 정교화 멀티에이전트 패턴. 결과를 단계적으로 개선한다.

### Spawn (하위 에이전트)
메인 세션에서 독립된 하위 에이전트를 실행하는 기능. 복잡한 작업을 병렬로 처리할 수 있다.

### TDD (Test-Driven Development)
테스트 주도 개발. 테스트 케이스를 먼저 작성하고 기능을 구현한다.

### TTL (Time To Live)
캐시 유효 기간. 기본 1시간으로 설정한다.
